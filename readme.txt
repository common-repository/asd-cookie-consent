This plugin showns users that your site uses cookie.

== Description ==
This plug-in shows users that your site uses cookie.
The choice of colours and the configuration of the various sections are easy and intuitive.
You can quickly have your message properly functioning on your website.

== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/asd-cookie-consent" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen in WordPress
3. Use the Settings->ASDCookieConsent screen to configure the plugin
